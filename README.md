# Practice: Weather Dashboard with API Request

You're building a morning routine app that helps users plan their day by showing current weather conditions for their city using a free weather API. Build a Java console application that fetches real weather data and displays a simple dashboard.

## Core Features

1. **Current Weather** – Get current conditions for a specific city  
2. **Weather Summary** – Display temperature, description, and humidity in a user-friendly format  
3. **Multiple Cities** – Allow user to check weather for 2-3 different cities  
4. **Simple Menu** – Console menu to choose which city to check  

## Technical Specs

- Use [OpenWeatherMap API (free)](https://openweathermap.org/api)
